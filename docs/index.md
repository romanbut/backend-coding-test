#Backend coding exercise for r.but

### 1. General info
Simple application that allows retrieval and creation of [`Ride`](./api/rides.md#Scheme) entity  

### 2. Technical specifications
1. Requirements:
   - `node (>8.6 and <= 10)` and `npm` are installed
2. Installation:
   - Run `npm install` 

### 3. API details

- [Rides](./api/rides.md)

- [Util](./api/utils.md)

### 4. Scripts

To run project:
`npm run start`

To run tests:
`npm run test`
